### Controll with keyboard
> <kbd>SPACE</kbd> - start/pause simulation <br>
> <kbd>R</kbd> - restart <br>
> <kbd>&#8593;</kbd> - increase speed<br>
> <kbd>&#8595;</kbd> - decrease speed<br>
